# Frontend

The frontend is a static app served by the cross-platform dev controller.

Default pages:

- App: `http://127.0.0.1:3000`
- Admin: `http://127.0.0.1:3000/admin.html`

Start or stop the full stack from the repository root:

```bash
python dev.py start
python dev.py stop
```

The frontend reads `frontend/assets/runtime-config.js`, which is generated by `dev.py` from `.env`.
